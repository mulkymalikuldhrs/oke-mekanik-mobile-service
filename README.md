## 🚀 Vision
OKE MEKANIK adalah platform layanan jasa mekanik panggilan terdepan yang menggabungkan kemudahan interaksi ala super-app (Uber/Grab) dengan teknologi diagnosa kendaraan tercanggih. Kami menghadirkan solusi perbaikan tepat waktu langsung di lokasi Anda (On-Time at Location).

## 💎 Key Features (v5.8.1)

### 🧠 AI Neural Diagnostic v5.8.1
Engine diagnosa cerdas yang memahami istilah teknis dan slang otomotif Indonesia (seperti *brebet, ngelitik, ngobos, ngeden*). Memberikan estimasi penyebab masalah dan tingkat urgensi secara instan.

### 📍 Real-Time Tracking & Maps
Integrasi peta interaktif untuk melacak posisi mekanik secara real-time. Rasakan kemudahan memantau perjalanan mekanik menuju lokasi Anda dengan presisi tinggi.

### 💰 Transparent Pricing (Uber-Style)
Sistem perhitungan biaya yang jujur dan transparan:
- **Jasa Perbaikan:** Biaya jasa sesuai standar layanan.
- **Suku Cadang:** Harga sparepart bengkel yang kompetitif.
- **Service Fee:** Biaya platform yang terjangkau.
- **Dynamic Fee:** Penyesuaian biaya otomatis untuk layanan darurat (Emergency).

### 🛠️ Professional Ecosystem
- **Expertise Matching:** Algoritma pencocokan yang menghubungkan Anda dengan mekanik tersertifikasi sesuai bidang masalah (Mesin, Kelistrikan, Kaki-kaki, dll).
- **Service Timeline:** Feed sosial yang menampilkan aktivitas layanan terkini dan ulasan nyata dari komunitas.
- **Secure Communication:** Fitur Chat dan Telepon langsung antar pengguna untuk koordinasi yang lebih baik.

## 🛠️ Tech Stack
- **Frontend:** React 19, Vite 8, Tailwind CSS (Futuristic Glassmorphism UI), Framer Motion.
- **Backend:** Node.js (Express 5), Socket.io (Real-time events).
- **Database:** SQLite (Better-SQLite3) for persistent, high-performance data storage.
- **Reliability:** Zod validation, JWT authentication, and Helmet security.

## 🚦 Getting Started

### Prerequisites
- Node.js (Latest stable version)
- NPM

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
3. Start the development server (Frontend & Backend):
   ```bash
   npm run dev
   ```
4. Access the application at `http://localhost:8080`.
