# 🔧 Oke Mekanik - Masterpiece v28.1 (v5.8.1 ULTIMATE+)

Oke Mekanik is a futuristic, high-fidelity mobile mechanic platform designed to bring professional vehicle services directly to the customer's location.

## 🚀 Features
- **AI Diagnostic Engine v5.8.1**: Advanced neural mapping for Indonesian automotive symptoms.
- **Real-time Tracking**: Monitor mechanic location and status via Socket.io.
- **Glassmorphism UI**: High-fidelity futuristic aesthetic with v28.1 standards.
- **Zero-Mock Policy**: 100% real data persistence via Express 5.2.1 and SQLite.
- **PWA Ready**: Offline-first capabilities with specialized caching strategies.
- **Multi-language Support**: Fully localized for Indonesian and English.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite 8, Tailwind CSS, Framer Motion.
- **Backend**: Express 5.2.1, Better-SQLite3, Socket.io.
- **Testing**: Vitest 4, Playwright.

## 📦 Getting Started
1. **Install Dependencies**:
   ```bash
   npm install --legacy-peer-deps
   ```
2. **Start Ecosystem**:
   ```bash
   npm run dev
   ```
   *Starts both Vite (8080) and Express (3001) concurrently.*
