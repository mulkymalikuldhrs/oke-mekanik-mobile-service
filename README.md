# OKE MEKANIK MASTERPIECE v27 - Full-Stack Production Ecosystem

Oke Mekanik is a revolutionary on-demand mobile mechanic platform, now upgraded to **Masterpiece v27**. This ecosystem features a hardened Node.js/Express backend, a high-intensity Glassmorphism frontend, and AI-powered vehicle diagnostics.

## 🚀 Key Features

- **Masterpiece v27 UI**: High-intensity Glassmorphism (>40px blur), holographic glow effects, and Framer Motion orchestrations.
- **Full-Stack Integration**: Real Node.js backend with SQLite persistence via `better-sqlite3`.
- **Real-Time Communication**: Socket.io integration for instant status updates and chat.
- **AI Smart Diagnostic v4.0**: Advanced weighted analysis engine for Indonesian vehicle technical scenarios.
- **Production Hardened**: JWT authentication, bcryptjs hashing, Helmet security, and Rate Limiting.
- **Full Localization**: Seamlessly toggle between Bahasa Indonesia (ID) and English (EN).
- **Comprehensive Testing**: Centralized Vitest suite and Playwright E2E verification.

## 🛠 Tech Stack

- **Frontend**: React 19, Vite, TypeScript, Tailwind CSS, shadcn/ui.
- **Backend**: Express 5.2.1, SQLite, Socket.io, Zod.
- **Security**: JWT, bcryptjs, Helmet, Express Rate Limit.
- **Testing**: Vitest, React Testing Library, Playwright.

## 🏁 Getting Started

### Prerequisites
- Node.js v22+
- npm v10+

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the full-stack environment:
   ```bash
   npm run dev
   ```

### Access Points
- **Frontend**: `http://localhost:8080` (or `5173` depending on environment)
- **Backend API**: `http://localhost:3001`

## 🧪 Testing

Run the centralized unit test suite:
```bash
npm test
```
